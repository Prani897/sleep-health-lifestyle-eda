   # Data Directory
   
   This folder contains the dataset files.
   
   ## Files:
   - `sleep_health_dataset.csv` - Original dataset
   - `sleep_health_processed.csv` - Cleaned and processed dataset
   - `person1_sleep_data.csv` - Sleep analysis subset
   - `person2_lifestyle_data.csv` - Lifestyle analysis subset
   - `person3_health_data.csv` - Health metrics subset
   - `person4_full_data.csv` - Complete dataset with all features
   
   ## Source:
   Kaggle - Sleep Health and Lifestyle Dataset
   https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset
